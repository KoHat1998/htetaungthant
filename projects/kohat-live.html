Kohat-Live

Real-time WebRTC conferencing with multi-room, pin, and mediasoup backend.

Open live app View on GitHub

Overview

Kohat-Live is a lightweight video conferencing web app optimized for low-latency group calls. Users can create rooms, invite others via room ID, pin streams locally, and share multiple tracks.

Key features

  • Multi-room architecture (independent rooms and IDs)
  • Local pin: switch main view without affecting others
  • Mediasoup SFU for scalable routing
  • ICE/STUN/TURN support for NAT traversal
  • Responsive UI with primary video + side thumbnails

Stack: Node.js, Express, Socket.IO, Mediasoup, HTML/CSS/JS, AWS EC2, Caddy/Nginx

Status: Live
